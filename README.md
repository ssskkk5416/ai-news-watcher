# ai-news-watcher
AI最新ニュースを分かりやすく閲覧するアプリケーション

## 画面設計（ワイヤー相当）
### ページ構成
- 一覧ページ
  - PDF一覧（タイトル/日付/発行元/概要のサマリ）
  - 検索・フィルタ（発行元、日付範囲、タグ）
  - 一覧の各行から詳細ページへ遷移
- 詳細ページ
  - PDFビュア（本文の閲覧）
  - メタデータ（タイトル、日付、発行元、要約、タグなど）
  - 前後のPDFへのナビゲーション

### ワイヤー（テキスト）
```
[一覧ページ]
----------------------------------------
 ヘッダー / 検索・フィルタ
----------------------------------------
 タイトル | 日付 | 発行元 | 概要
  └ クリックで詳細ページへ
----------------------------------------

[詳細ページ]
----------------------------------------
 タイトル / 日付 / 発行元 / タグ / 要約
----------------------------------------
 [PDFビュア]
----------------------------------------
 前後ナビゲーション
```
