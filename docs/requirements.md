# 要件メモ（最小機能）

## PDFの登録方法
- 手動アップロード（管理画面または一覧ページからアップロード）
- 監視フォルダの自動取り込み（指定フォルダへの配置を検知）

## 一覧表示
- 表示項目
  - タイトル
  - 日付（発行日/登録日）
  - 発行元
  - 概要（要約の短縮版）
  - タグ（任意）

## PDFビュアの配置
- 一覧→詳細ページへ遷移して表示
- 詳細ページ内でPDFビュアをメタデータの下部に配置

## 取得すべきメタデータ項目
- ファイル名
- タイトル
- 発行日（もしくは日付）
- 登録日
- 発行元
- 要約（本文からの自動生成）
- タグ（手動付与/自動抽出）
- ページ数
- ソースURL（任意）
